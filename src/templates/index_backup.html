Umbrella App

Heading out? Wondering if you should bring an umbrella? This app will keep you dry.

Virtual crossing weather API provides the chance of rain. Twilio sends an SMS message letting you know whether you need an umbrella. The styling by tailwindcss and the background image by ChatGPT 4o make this site look great whether your using a computer or your phone.

This Pthon Flask app is an enhanced version of the rain tracker app in Angela Yu's 100 Days of Code: The Complete Python Pro Bootcamp.

Dad Jokes

This project is from 50 Projects in 50 Days - HTML, CSS, and JavaScript by Brad Traversy, Florin Pop.

I'm still early in my journey to learn css and JavaScript. The css and js here is taken directly from the step by step project instructions.

To publish this on the web, I used my mad Python skills to create a Flask app. For practice, I rewrote the JavaScript portion using the Requests module. In this version of Dad Jokes v2 the page reloads each time you request a new joke.

A special thanks to Angela Yu's teaching in 100 Days of Code: The Complete Python Pro Bootcamp, which enabled me to develop my Python skills.
